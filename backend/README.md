# EventHub Backend

Backend API for EventHub - A social events and plans platform.

## Features

- User authentication and authorization
- Personal and organizer profiles
- Events management with approval system
- Social plans creation
- Feed with featured events
- Admin dashboard
- File upload with Cloudinary
- Email notifications

## Tech Stack

- Node.js with TypeScript
- Express.js
- PostgreSQL with TypeORM
- JWT for authentication
- Cloudinary for file storage
- Nodemailer for emails

## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL 15+
- npm or yarn
